# Projects

Navigation: [[index]] | [[areas/_index]] | [[resources/_index]] | [[archives/_index]]

Active work with a defined outcome and (usually) a deadline. Each project gets its own subfolder: `wiki/projects/<project-name>/`.

New, unsorted ingests and session notes land in [[projects/inbox/_index|inbox]] for triage — move them into a named project folder or into [[../areas/_index|areas]] / [[../resources/_index|resources]] once you know what they are.

## Active

- [[segundo-cerebro/README|Segundo Cerebro]] — página principal del proyecto: objetivo, flujo `.raw/` → `wiki/` → `hot.md` → commit → GitHub, herramientas, carpetas, próximos pasos (status: active)

## Inbox (untriaged)

- `wiki/projects/inbox/` — new session notes and ingests file here by default under PARA mode. Empty until first use.

---

When a project finishes (shipped, cancelled, or superseded), move its folder to `wiki/archives/<year>/` and remove it from the Active list above.
